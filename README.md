# GoGoGo

GoGoGo是一款敏捷项目管理软件/工具集

## 主要设计目标：
1. 内建账号登录
2. 产品库
3. 项目库
4. 任务板
5. 微信登录
6. 任务上下文图文消息
7. 任务赏金系统
8. 源码版本管理集成
9. 测试平台集成
10. VS任务插件集成
11. AI辅助功能


GoGoGo is an agile project management software/tools

1. built-in account login
2. product library
3. project library
4. task board
5. WeChat login

9. test platform integration
